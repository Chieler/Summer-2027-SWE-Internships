# cutoff, so you never lose track of something you've already submitted.

#   1. After you apply to a role, copy its Apply link from the README.
#   2. Open this file on GitHub, tap edit, paste the link on a new line.
#   3. Commit. Next run marks it ✅.
#

# and partial/substring matches work (a job ID is enough).
